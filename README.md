# Dino2D

## Overview
Dino2D is a 2D game project developed using Python, VS Code, and Tiled for level design.

## Note
This project cannot currently be initialized because it is **out of date** with the current engine and development environment.
- Updates are currently on the way.
  
## Development Environment

- **Version Control:** GitHub / GitHub Desktop  
- **Code Editor:** Visual Studio Code  
- **Database:** MongoDB  
- **Level Creator:** Tiled  
- **Programming Language:** Python  
- **Sprites:** Itch.io
